  • Sobre

    Este projeto foi idealizado com o objetivo de criar uma plataforma para a configuração de switches Datacom L2/L3 de modo a auxiliar e aumentar a produtividade
    do setor NOC responsável pela configuração e monitoramento de dispositivos de rede diversos. Dessa forma, esta aplicação fornece um sistema que possibilita a
    configuração de variados switches de forma mais rápida e automatizada, sem a necessidade de muito conhecimento técnico do processo, fornecendo uma interface
    de usuário amigável para o procedimento. Dentre as funcionalidades deste projeto está a capacidade de configurar de forma massiva múltiplos switches com todos
    os elementos necessários para deixá-los prontos para uso imediato.
